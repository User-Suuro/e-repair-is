Extract Guna UI

Run Keygen.exe in GunaUIKeyGen

Go to project settings, then locate namespace

Add Guna.UI and Guna.UI2 .dll

Also add rldc report viewer .dll to namespace if it is gone

Uncheck "Allow parallel project initialization" in Tools -> Options -> Project and Solutions -> General

If you want to use GunaUI:

Make a category in toolbox

Right click the created category, then click choose items

Then browse for GunaUI.dll

For Image Path Data:

Download Image Data: https://drive.google.com/drive/folders/1r0LiezDCpt8i1_wKPT87SMssfBCkShrx?usp=sharing

Then Go to C:\Users\User\AppData\Roaming

Place E-Repair Images there
